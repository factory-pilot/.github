---

Factory Pilot is a real-time anomaly detection platform for manufacturing. We bring machine learning to the shop floor without requiring any data science expertise -- connect your machines, and the platform detects deviations as they happen and drives the right action. Cut scrap, avoid downtime, ship on time.

**Connect** -- OPC UA, MTConnect, MQTT, Modbus/TCP, REST. Edge gateway normalizes signals from any PLC, CNC, robot, or sensor.

**Detect** -- Thresholds, drift, rate-of-change, SPC rules, and adaptive ML models. Sub-second detection, zero configuration.

**Act** -- Role-based alerts and workflows via Teams, Slack, Email, SMS, and webhooks. Fully auditable escalations and CAPA integration.

### From this GitHub

Factory Pilot itself is not open source. Here we publish tools and utilities for the manufacturing data community.

**[test-historians](https://github.com/factory-pilot/test-historians)** -- Docker environment that spins up TimescaleDB, InfluxDB, and Grafana with 7 days of synthetic sensor data in a single command. Built for anyone developing against time-series databases.
